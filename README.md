# Overview

This program is a simple shooter, meant not only for practice with python classes and the pygame library, but also for practice with building a game loop

Simple point and click shooter. Move player with WASD, aim with mouse, fire with LMB

I wanted to challenge myself to create a game that required calculating angles and collisions

[Software Demo Video](https://youtu.be/Ezm5s_OFYPU)

# Development Environment

Created in VS code, with quality of life extensions. Player, zombie, and bullet images created with MS paint

Created in python. Libraries used were Pygame, Sys, Math, and Random.
